# Claude Code Marketplace

Plugins that extend Claude Code with powerful development workflows.

## Installation

```bash
git clone git@github.com:SavvyAI/claude-code-marketplace.git
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [Pro](./pro/readme.md) | Professional development workflows with planning, branch management, and PR automation |

## Contributing

1. Fork this repository
2. Create your plugin in a new directory
3. Add a `commands/` folder with `.md` command files
4. Submit a pull request
